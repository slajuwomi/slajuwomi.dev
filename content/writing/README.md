# Writing content

Add published posts to this folder as `.mdx` files. The writing index ignores this README.

Each post needs frontmatter:

```mdx
---
title: "Post title"
date: "2026-07-10"
description: "A short search and social description."
---

Post content starts here.
```

The filename becomes the URL slug. For example, `learning-in-public.mdx` becomes `/writing/learning-in-public`.
